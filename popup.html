Sea of Thieves API Helper

This extension helps retrieve the 'rat' cookie from the Sea of Thieves website.
This cookie is required to use the Sea of Thieves fAkePI.
Developed by Perryhdlp, GitHub

  1. Navigate to Sea of Thieves and log in
  2. Click the "Retrieve Cookie" button to get the cookie value
  3. Use the "Copy to Clipboard" button to copy the value
  4. Navigate to sotfAkPI and paste the cookie value into the "Cookie" field
